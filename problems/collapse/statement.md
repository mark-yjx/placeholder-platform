# Collapse Identical Digits

A sequence of identical digits is collapsed to one digit
in the returned integer.

You can assume that the function is called with an integer as argument.

## Input

A single integer. The value may be positive, negative, or zero.

## Output

Return the integer produced by collapsing each run of adjacent identical
digits to one digit while preserving the original sign.
