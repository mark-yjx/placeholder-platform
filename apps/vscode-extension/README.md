# OJ VSCode Extension

Minimal VS Code extension package for the COMP9021 practice platform.

## Versioning
- Releases use Semantic Versioning.
- Current packaged release: `0.1.0`

## Commands
- `OJ: Login`
- `OJ: Fetch Problems`
- `OJ: Submit Code`
- `OJ: View Result`
- `OJ: Favorite Problem`
- `OJ: Submit Review`
- `OJ: Show Public Stats`
- `OJ: Show Public Ranking`
